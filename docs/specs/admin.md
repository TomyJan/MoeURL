# 管理后台

## 目标

管理员后台用于系统级管理。

v0.0.1 只要求提供管理员查看和管理全站短链的最小入口，以及首次初始化流程。完整后台能力按后续版本逐步实现。

## 后台导航结构

完整目标形态建议按二级分组组织：

```text
系统仪表盘

域名与链接
  链接管理
  域名管理

用户与权限
  用户管理
  用户组管理

系统维护
  系统设置
  主题设置
  审计日志 / 操作记录
```

权限不建议作为独立的常规菜单长期存在。权限配置应主要内聚在用户组编辑页面中，便于管理员理解「用户组 → 权限」的关系。

管理后台可以与用户中心共用同一套界面，只在菜单中展示管理入口。

## 系统仪表盘

系统仪表盘用于展示全站概况。

建议包含：

- 用户总数。
- 短链总数。
- 今日新增短链。
- 今日访问量。
- 近期访问趋势。
- 系统状态摘要。

系统仪表盘不属于 v0.0.1 范围。

## 域名与链接

链接管理用于管理全站短链。

管理员可以：

- 查看全部短链。
- 搜索全部短链。
- 按用户、域名、状态筛选。
- 禁用短链。
- 删除短链。
- 修改短链所属用户。
- 查看短链统计。

v0.0.1 要求提供管理员查看和管理全站短链的最小入口。该入口至少支持查看全站短链列表、识别所属用户、启用或禁用短链、软删除短链。搜索和多维筛选可以作为增强项，不作为 v0.0.1 阻塞验收。

域名管理用于管理系统可用域名。

管理员可以：

- 添加域名。
- 编辑域名。
- 禁用域名。
- 删除域名。
- 设置全局默认域名。
- 配置域名可用用户组。
- 配置域名默认跳转方式和默认跳转页模板。
- 配置域名排序。

完整域名管理页面不属于 v0.0.1 范围。

## 用户与权限

用户管理用于维护本地用户资料和账号状态。

管理员可以：

- 查看用户列表。
- 搜索用户。
- 查看用户详情。
- 启用或禁用用户。
- 调整用户组。
- 查看用户短链数量。
- 查看用户访问量。

用户组管理用于配置权限。

管理员可以：

- 查看用户组。
- 创建用户组。
- 编辑用户组。
- 配置用户组权限。
- 配置用户组可用域名。
- 配置用户组可用跳转方式和模板。
- 配置用户组默认域名。
- 查看用户组成员。

内置用户组不可删除，但可以修改部分权限。

完整用户与权限管理页面不属于 v0.0.1 范围。

v0.0.1 不实现完整用户管理页面，但管理员必须具备创建普通用户的最小入口。该入口仅支持设置用户名、密码、昵称、用户组和启用状态，用于满足普通用户登录和创建短链的闭环。用户列表、用户详情、用户删除、批量导入、多用户组和权限可视化配置均属于后续能力。

## 用户组编辑页目标结构

用户组编辑页建议按能力分组展示，避免权限项平铺导致难以理解：

```text
用户组基础信息
  名称
  标识
  描述
  是否内置

短链能力
  创建短链
  自定义短码
  设置标题、备注、标签
  设置有效期
  使用访问密码

域名权限
  可用域名
  默认域名

跳转能力
  直接跳转
  中间页跳转
  确认页跳转
  可用跳转页模板
  可编辑模板元数据

统计权限
  基础统计
  维度统计
  详细统计
  导出统计

管理权限
  用户管理
  用户组管理
  全站短链管理
  域名管理
  系统设置
  主题设置
  审计日志
```

权限包和权限预设可以作为用户组编辑页中的快捷配置入口，但最终仍应落到具体权限项。

## 系统维护

系统维护用于管理全局配置、主题和审计记录。

系统设置建议包含：

- 默认语言。
- 默认主题。
- 默认短码长度。
- 默认跳转方式。
- 默认中间页倒计时。
- 默认短链有效期。
- 默认用户组。
- 本地登录开关（需存在可用外部登录方式时才允许关闭）。
- 外部接口鉴权扩展入口，用于后续支持程序化创建短链、查询短链和获取统计数据。
- OIDC 提供商管理。
- OIDC 提供商预置模板。

主题设置建议包含：

- 站点名称。
- Logo。
- Favicon。
- 主题色。
- 首页文案。
- 页脚内容。
- 默认跳转页模板。

审计日志 / 操作记录用于查询关键操作历史。

具体功能是否可用，仍应由权限系统最终决定。

## 首次初始化流程

首次初始化流程属于 v0.0.1 必须实现能力。

初始化内容至少包括：

- 注册第一个用户，该用户加入内置 `admin` 用户组。
- 设置站点名称。
- 配置默认系统访问域名或确认当前访问域名。
- 配置默认短链访问域名。
- 选择默认语言。
- 写入固定内置权限模型。

初始化完成前，普通登录、短链创建和管理后台应不可用。初始化完成后，系统应写入固定内置权限模型，并禁止重复执行初始化流程，除非管理员通过明确的维护机制重置系统状态。
