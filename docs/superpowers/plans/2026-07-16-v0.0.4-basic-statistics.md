# v0.0.4 基础统计前置实现计划

> **面向 AI 代理的工作者：** 必需子技能：使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 逐任务实现此计划。步骤使用复选框（`- [ ]`）语法来跟踪进度。

**目标：** 为 MoeURL v0.0.4 实现短链访问事件入库和列表基础统计展示。

**架构：** 新增 `short_link_event` 表保存成功跳转事件；`internal/event` 提供数据库 recorder；短链列表 SQL 聚合访问量并通过既有列表 API 返回；前端控制台列表展示统计字段。

**技术栈：** Go、Chi、SQLC、Goose、PostgreSQL、Vue 3、TypeScript、Vuetify 4、Vitest、Playwright。


预期：FAIL，提示 `short_link_event` 不存在。

- [ ] **步骤 2：新增 migration**

创建 `migrations/00002_short_link_event.sql`，包含：

```sql
-- +goose Up
-- +goose StatementBegin
create table short_link_event (
    id uuid primary key,
    short_link_id uuid not null references short_link(id),
    event_type text not null,
    created_at timestamptz not null
);

create index short_link_event_short_link_id_idx on short_link_event(short_link_id);
create index short_link_event_type_created_at_idx on short_link_event(event_type, created_at);
create index short_link_event_short_link_created_at_idx on short_link_event(short_link_id, created_at);
-- +goose StatementEnd

-- +goose Down
-- +goose StatementBegin
drop table if exists short_link_event;
-- +goose StatementEnd
```

- [ ] **步骤 3：运行测试验证通过**

运行：

```bash
go test ./internal/db -run TestMigrations -count=1
```

预期：PASS。
