# v0.0.3 配色与交互重设实现计划

> **面向 AI 代理的工作者：** 必需子技能：使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 逐任务实现此计划。步骤使用复选框（`- [ ]`）语法来跟踪进度。

**目标：** 将已确认的「雾蓝石墨」配色与交互设计固化到前端默认主题中，使实际页面视觉尽量对齐 `docs/superpowers/prototypes/v003-color-interaction-redesign.html`。

**架构：** 主题颜色仍集中在 Vuetify theme token 和 `theme-css.ts` 全局 CSS token 中。组件级调整只覆盖偏好切换和控制台多级导航，不改变页面路由、业务 API、权限模型和数据流。

**技术栈：** Vue 3、TypeScript、Vuetify 主题系统、Vitest、Vue Testing Library。
