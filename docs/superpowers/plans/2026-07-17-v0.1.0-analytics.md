# v0.1.0 短链统计分析实现计划

> **面向 AI 代理的工作者：** 必需子技能：使用 `superpowers:subagent-driven-development`（推荐）或 `superpowers:executing-plans` 逐任务实现此计划。步骤使用复选框（`- [ ]`）语法来跟踪进度。

**目标：** 将 v0.0.4 的列表级访问指标扩展为受权限隔离、隐私最小化的短链分析页。

**架构：** Redirect handler 在确认跳转响应成功后提取经过归类的请求元数据，交给现有有界异步 recorder 持久化。`shortlink` 模块使用 SQLC 聚合事件表并通过用户与管理员分路由提供统计结果；Vue 页面复用现有控制台 Shell 与 TanStack Query 展示概览、7 日趋势和维度聚合。

**技术栈：** Go 1.25、Chi、SQLC、Goose、PostgreSQL、Vue 3、TypeScript、Vuetify 4、TanStack Query、Chart.js、Vitest、Playwright。

---

## 文件结构

- `migrations/00003_short_link_event_dimensions.sql`：增加经最小化处理的事件维度及聚合索引。
- `queries/event.sql`：定义趋势和维度聚合 SQLC 查询。
- `internal/db/sqlc/event.sql.go`：由 `sqlc generate` 生成的查询类型和方法。
- `internal/event/event.go`、`recorder.go`：将分类后的元数据写入现有异步 recorder。
- `internal/config/config.go`、`internal/app/app.go`：配置并注入可选地区请求头名称。
- `internal/shortlink/analytics.go`：请求元数据解析与设备归类。
- `internal/shortlink/model.go`、`service.go`、`handler.go`：统计领域模型、权限服务和 HTTP API。
- `internal/http/router.go`：注册两条统计 API。
- `web/src/entities/short-link/{model,api}.ts`：前端统计契约和请求函数。
- `web/src/pages/AnalyticsPage.vue`：分析页及其加载、空态、错误态。
- `web/src/pages/{MyLinksPage,AdminLinksPage,ConsoleLinkList}.vue`：短链列表的分析入口。
- `web/src/app/{router,i18n}.ts`、`web/src/widgets/console-shell/ConsoleShell.vue`：替换占位路由和导航标识。
- `web/e2e/analytics.spec.ts`、`.github/workflows/code-check.yml`：E2E 主流程和 CI 门禁。
- `docs/product/`、`docs/implementation/`：v0.1.0 范围、任务、验收和技术基线同步。

## 计划自检

- 规格中的隐私边界、统计口径、权限、趋势、维度、前端状态和 E2E 都有独立任务覆盖。
- 所有新增类型均在对应任务中定义，用户和管理员接口的路径、权限与前端 scope 一致。
- 计划未包含密码、过期时间、二维码、访问明细或任意日期范围。
- 每项实现变更均先定义失败测试，再给出具体实现和验证命令。
