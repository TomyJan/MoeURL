# v0.0.3 任务级实施清单

> 本文档把 [v0.0.3 实施计划](./v0.0.3-plan.md) 拆成可执行任务。具体编码时，应继续遵循技术基线、v0.0.3 UI/UX 规格和覆盖率门禁。

## 1. 当前状态

v0.0.2 已完成发布后质量补强闭环。当前 v0.0.3 聚焦 UI/UX 精细化重设计，不新增后端核心业务能力。

v0.0.3 的优先级顺序是：

1. 文档和设计基线同步。
2. 主题 token 与全局视觉系统。
3. 短链生成组件复用。
4. 首页重设计。
5. 控制台 Shell 和页面迁移。
6. 响应式、测试、E2E 和验收收尾。
