# v0.1.0 验收清单

> 未执行的项目不得标记为完成。

## 数据与隐私

- [ ] migration 可创建和回滚事件维度字段。
- [ ] 只保存来源主机名、设备类型和可选国家代码。
- [ ] 不保存原始 IP、完整 User-Agent 或完整 Referer。
- [ ] 仅成功写出跳转响应后写入统计事件。
- [ ] recorder 满载或失败不会阻断跳转。

## API 与权限

- [ ] 用户统计 API 只返回当前用户短链。
- [ ] 管理员统计 API 可返回任意未删除短链。
- [ ] 概览、连续 7 天趋势和三类维度聚合口径正确。
- [ ] 无效 ID、权限不足、短链不存在和基础设施错误符合既有响应约定。

## 控制台

- [ ] `/analytics` 展示真实短链数据，不展示伪造数据。
- [ ] 两个短链列表可进入对应分析页。
- [ ] 页面具备加载、空数据、错误和移动端状态。
- [ ] 中英文文案完整，分析导航不再标记为规划中。

## 发布验证

- [ ] `go test ./...` 通过。
- [ ] `go vet ./...` 通过。
- [ ] 后端覆盖率门禁为 100%。
- [ ] `cd web && pnpm lint`、`pnpm typecheck`、`pnpm test`、`pnpm test:coverage`、`pnpm build` 通过。
- [ ] `cd web && pnpm test:e2e` 在 CI 通过。
- [ ] `git diff --check` 通过。
