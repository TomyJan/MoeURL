> 本文档用于 v0.0.2 开发完成后的功能验收。未经验证的项目不得标记为完成。

- 优先验证 v0.0.2 新增能力和 v0.0.1 基础闭环回归。

- 自动化验证应记录命令和结果。
- 手工验证应记录环境、步骤和结果。
